## Request/Response Flow Examples

### Example 1: Get Insights
```
REQUEST:
  GET /api/ai/insights

PROCESSING:
  1. Route receives request
  2. Calls getAnalytics() from database
  3. Formats data into AI prompt
  4. Calls OpenAI API with system prompt + analytics
  5. Receives structured insights from Claude

RESPONSE:
{
  "insights": [
    {
      "title": "Q4 Pipeline Surge",
      "description": "...",
      "recommendation": "...",
      "priority": "HIGH"
    }
  ],
  "analytics": { ... }
}

RENDERING:
  1. Frontend receives JSON
  2. Maps over insights array
  3. Renders insight cards with colors
  4. Shows title, description, recommendation, priority
```

### Example 2: Generate Email
```
REQUEST:
  POST /api/ai/email-template
  {
    "customerId": 1,
    "dealId": 5,
    "templateType": "followup"
  }

PROCESSING:
  1. Route validates parameters
  2. Fetches customer and deal from database
  3. Formats context into AI prompt
  4. Includes template type instructions
  5. Calls OpenAI API
  6. Receives email subject + body

RESPONSE:
{
  "subject": "Following Up on Your Q4 Initiative",
  "body": "[Professional email content]"
}

RENDERING:
  1. Frontend displays preview
  2. Shows subject line
  3. Shows email body formatted
  4. Provides copy-to-clipboard button
  5. User can paste into email client
```
